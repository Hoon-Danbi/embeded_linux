/* MDS2450 보드 설정값 */

터미널서 다음과 같이 명령어를 입력한다

user@linux:~$ bash ~/.getBootEnv 192.168.0.2 fw2

set bootargs 'mem=500M root=/dev/nfs rw nfsroot=192.168.0.2:/usr/local/arm/4.2.2-eabi/usr/sys-root,nolock,tcp ip=192.168.0.3:192.168.0.2:192.168.0.1:255.255.255.0::eth0:off console=ttySAC0,115200n81'

set bootdelay 1
set gatewayip 192.168.0.1
set netmask 255.255.255.0
set ipaddr 192.168.0.3
set serverip 192.168.0.2
set bootcmd 'tftp 30000000 MDS2450.bin;go 30000000'
saveenv

user@linux:~$ 

화면 내용을 캡쳐하여 minicom 상에서 입력하여 사용한다
